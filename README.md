# customer_behavior_analysis
data analyst project showcasing customer behavior analysis using python SQL and power BI. 

Data Analytics Project
1. Overview

This project demonstrates an end-to-end data analytics workflow. The objective is to analyze a dataset, extract meaningful insights, and present the results through interactive dashboards and reports.

The project involves data loading, exploratory data analysis (EDA), data cleaning, SQL querying, visualization, and reporting. Python is used for data processing, SQL databases are used for querying structured data, and Power BI is used to build an interactive dashboard for decision-making.

2. Dataset

The dataset contains structured data used to analyze patterns and trends related to business performance.

Key characteristics:

Multiple columns representing categories, transactions, and metrics

Contains missing values and inconsistencies that require cleaning

Suitable for performing aggregation, filtering, and trend analysis
